# Getting Started

## Requirement

```
xcode-select --install
```

## Install

```
curl https://raw.githubusercontent.com/blcsntb/mac-setup/dotfiles/dotfiles.sh | bash
source ~/.dotfiles/dotfiles.sh
install_dotfiles
```

## Upgrade

```
upgrade_dotfiles
```
